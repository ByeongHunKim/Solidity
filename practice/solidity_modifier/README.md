# 모디파이어 종류

## view 
- function 밖의 변수들을 읽을 수 있으나 변경 불가

## pure 
- function 밖의 변수들을 읽지 못하고 변경 불가

## 둘 다 명시하지 않을 때
- function 밖의 변수들을 읽어서 변경해야 한다.
